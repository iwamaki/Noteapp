# Deep Work

**Author:** Cal Newport
**Rating:** ★★★★☆

## Summary

Deep work = Professional activities performed in a state of distraction-free concentration.

## Key Concepts

### The Deep Work Hypothesis

> Deep work is becoming increasingly rare and increasingly valuable in our economy.

### Strategies

1. **Monastic** - Eliminate all distractions
2. **Bimodal** - Dedicated deep work periods
3. **Rhythmic** - Daily deep work habit
4. **Journalistic** - Fit deep work when possible

## Implementation

| Time | Activity |
|------|----------|
| 6:00 - 8:00 | Deep work |
| 8:00 - 12:00 | Meetings |
| 13:00 - 15:00 | Deep work |
| 15:00 - 17:00 | Shallow work |

## Takeaway

Schedule deep work blocks. Protect them fiercely.
