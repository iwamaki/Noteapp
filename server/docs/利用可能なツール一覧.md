# 利用可能なツール一覧

## 📝 ファイル操作ツール

### 1. create_file - ファイル作成
- 新しいファイルを作成します
- パラメータ：
  - title（必須）：ファイル名
  - content（オプション）：初期内容
  - category（オプション）：カテゴリー（階層形式、例：「Work/Minutes」）
  - tags（オプション）：タグ（カンマ区切り、例：「important,urgent」）

### 2. read_file - ファイル読み込み
- ファイルの内容を読み込みます
- パラメータ：
  - title（必須）：ファイル名

### 3. edit_file - ファイル編集（全体）
- ファイル全体を編集します
- パラメータ：
  - title（必須）：ファイル名
  - content（必須）：新しい内容（全文）

### 4. edit_file_lines - ファイル編集（部分）
- ファイルの特定行を編集します（大きなファイルに効率的）
- パラメータ：
  - title（必須）：ファイル名
  - start_line（必須）：開始行番号（1から始まる）
  - end_line（必須）：終了行番号（1から始まる）
  - content（必須）：新しい内容

### 5. delete_file - ファイル削除
- ファイルを削除します
- パラメータ：
  - title（必須）：ファイル名

### 6. rename_file - ファイル名変更
- ファイルの名前を変更します
- パラメータ：
  - title（必須）：現在のファイル名
  - new_title（必須）：新しいファイル名

## 🔍 検索ツール

### 7. search_files - ファイル検索
- ファイルを検索します
- パラメータ：
  - query（必須）：検索クエリ
  - search_type（オプション、デフォルト：「title」）：
    - title - ファイル名で検索
    - content - ファイル内容で検索
    - tag - タグで検索
    - category - カテゴリーで検索

### 8. web_search - ウェブ検索
- インターネットで検索し、詳細情報を取得します
- パラメータ：
  - query（必須）：検索クエリ
  - max_results（オプション、デフォルト：5）：最大結果数（1-10）
  - fetch_details（オプション、デフォルト：3）：詳細を取得する結果数（0-5）

記録日時: 2025年
ファイル構造: フラット構造（ディレクトリ/パス概念なし）
分類方法: カテゴリーとタグで管理
